This project was done for a competition organised by Drishti-Grand Robo Pix. A line follower robot is a mobile robot designed to autonomously follow a predefined line on the ground. For this project we used IR sensors, PCB, Arduino, Solder wire and Machine, etc. We considered many of the possibilities which the bot can encounter. The Arduino will receive input from sensors and based on the sensor condition the case is decided, and the bot moves accordingly when output is given to the motor pins.

We can divide the project into three parts:
Coding,
Circuit designing,
Assembling the bot.

We coded the Arduino in C in the Arduino IDE software.
